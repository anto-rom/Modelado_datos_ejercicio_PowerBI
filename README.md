# Modelado_datos_ejercicio_PowerBI
Ejercicio para  The Power sobre la creación de un dashboard de Power BI avanzado

🧠 Data Model

The solution is based on a star schema, which improves query performance and analytical clarity.

Fact Tables

FactSales – sales transactions and revenue metrics

FactProduction – production performance indicators

Dimension Tables

Calendar – date hierarchy for time analysis

DimProduct – product information

DimProduct_Product – product hierarchy

DimChannel – sales channel classification

DimPromotion – promotional campaigns

DimStores – store and geographic information

DimLinea – product line classification

This structure allows efficient analysis across multiple business dimensions.
